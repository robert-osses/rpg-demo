# Planificación del Proyecto: RPG Battle Simulation

Este documento describe la visión general y los requisitos para el simulador de batallas RPG.

## Visión General
El objetivo es crear una experiencia de combate por turnos premium y visualmente atractiva, inspirada en los RPG clásicos pero con una estética moderna.

## Características Principales
- **Sistema de Combate**: Por turnos, con acciones como Atacar, Habilidades, Ítems y Defender.
- **Personajes**: Héroes y enemigos con estadísticas propias (HP, MP, Ataque, Defensa).
- **Interfaz de Usuario**: Menús dinámicos, barras de vida animadas y efectos visuales impactantes.
- **Efectos Visuales**: Micro-animaciones para impactos, uso de habilidades y transiciones de turno.

## Tecnologías
- HTML5 / CSS3 (Vanilla CSS con diseño premium).
- JavaScript moderno (ES6+).
- Vite para el desarrollo y empaquetado.

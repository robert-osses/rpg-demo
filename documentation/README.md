# RPG Battle Simulation

Simulador de encuentros de combate RPG con un enfoque en la excelencia visual y la jugabilidad fluida.

## Documentación
Puedes encontrar la documentación detallada del desarrollo en la carpeta `documentation/`:
- [Planificación](planning.md)
- [Plan de Implementación](implementation_plan.md)

## Requisitos
- Node.js (v18 o superior)
- Navegador moderno

## Inicio Rápido
1. Instala las dependencias (cuando estén disponibles):
   ```bash
   npm install
   ```
2. Ejecuta el servidor de desarrollo:
   ```bash
   npm run dev
   ```
